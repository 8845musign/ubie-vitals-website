## Usage

- 垂直方向に、一定のリズムを持つレイアウト
  - リスト
  - フォームなど
- Figma 上で垂直方向の Auto Layout が指定されていた場合、`<Stack>` が役に立ちます
  - 後に、要素が増減したとしてもコード変更なしに対応することができます

## Accessibility

### `role` の指定が必要な場合

`<li>` を小要素に持つ場合、`<Stack>` は `role=list` である必要があります（大雑把な説明をすると、`<ul>`として振る舞うべきです）。その場合、`as` props を利用し、`<ul>` としてレンダリングするようにしてください。

```js
<Stack as="ul" spacing="md">
  <li>...</li>
</Stack>
```
